### 사용자 비밀번호 초기화

- 관리자ID로 로그인
- 상단 메뉴 `관리자기능`
- 좌측 메뉴 `담당자관리 - 담당자 정보관리`
- 대상 ID 검색하여 `수정` 버튼 클릭
- 좌측 하단 `비밀번호 초기화` 버튼 클릭
    - _비밀번호를 초기화 하시겠습니까?_ 예
    - _비밀번호 초기화 성공!_ 확인
- **( 중요! )** 좌측 하단 _`비밀번호 초기화 결과`_ 복사하여 사용자에게 통지
> 초기화된 비밀번호는 무작위 숫자 + 문자로 구성된 8자리 문자열이다.

### 비밀번호 입력 오류 횟수 초기화

- 관리자ID로 로그인
- 상단 메뉴 `관리자기능`
- 좌측 메뉴 `담당자관리 - 담당자 정보관리`
- 대상 ID 검색하여 `아이디`에 걸려있는 링크를 클릭
- `비밀번호 오류 횟수`칸에 있는 `초기화` 버튼을 클릭
